# 技術分析指標手冊

## 移動平均線 (Moving Average)

### 簡單移動平均線 (SMA)
簡單移動平均線是最基本的技術指標之一。計算公式為：

SMA(n) = (C₁ + C₂ + ... + Cₙ) / n

其中 C 為收盤價，n 為計算周期。

常用周期：
- 5日線（週線）：短期趨勢
- 20日線（月線）：中期趨勢
- 60日線（季線）：中長期趨勢
- 120日線（半年線）：長期趨勢
- 250日線（年線）：牛熊分界線

### 指數移動平均線 (EMA)
EMA 對近期價格賦予更高權重，反應更靈敏：

EMA(n) = C × k + EMA(yesterday) × (1-k)
其中 k = 2 / (n+1)

### 金叉與死叉
- **金叉**：短期均線上穿長期均線，為買入信號
- **死叉**：短期均線下穿長期均線，為賣出信號

## MACD (Moving Average Convergence Divergence)

MACD 由三部分組成：
1. **MACD線（DIF）**：12日EMA - 26日EMA
2. **信號線（DEA）**：MACD線的9日EMA
3. **柱狀圖（Histogram）**：MACD線 - 信號線

### MACD 交易信號
- **金叉買入**：DIF 上穿 DEA
- **死叉賣出**：DIF 下穿 DEA
- **零軸上方**：多頭市場
- **零軸下方**：空頭市場
- **頂背離**：價格新高但 MACD 未新高，看跌信號
- **底背離**：價格新低但 MACD 未新低，看漲信號

## RSI (Relative Strength Index)

RSI 衡量價格變動的速度和幅度：

RSI = 100 - 100/(1 + RS)
RS = 平均上漲幅度 / 平均下跌幅度

### RSI 應用
- RSI > 70：超買區域，可能回調
- RSI < 30：超賣區域，可能反彈
- RSI = 50：多空平衡

## 布林帶 (Bollinger Bands)

由三條線組成：
- 上軌：SMA(20) + 2×標準差
- 中軌：SMA(20)
- 下軌：SMA(20) - 2×標準差

### 布林帶應用
- 價格觸及上軌：可能超買
- 價格觸及下軌：可能超賣
- 帶寬收窄：波動率降低，可能出現大行情
- 帶寬擴大：波動率增加，趨勢可能持續

## KDJ 隨機指標

KDJ 由三條線組成：
- K線：RSV的3日移動平均
- D線：K線的3日移動平均
- J線：3K - 2D

### KDJ 應用
- K > 80 且 D > 80：超買
- K < 20 且 D < 20：超賣
- K上穿D：金叉買入
- K下穿D：死叉賣出
